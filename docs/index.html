Tiler

Draw the layout of your windows in a fast, cool manner.

macOS 13+ Free & open source
Download GitHub

How it works

1
Drag
Start dragging a window like you usually would
2
Trigger
Press Space (or your chosen trigger) — the grid appears
3
Draw
Draw the area you want the window to fill
4
Snap
Release — the window snaps instantly

Install

Homebrew

brew install VlatkoMilisav/tap/tiler

Manual

Download Tiler.dmg, open it, and drag Tiler into Applications.

The app isn't notarized because $99/year for an Apple developer certificate is a scam. macOS will complain — go to System Settings → Privacy & Security and click "Open Anyway".
